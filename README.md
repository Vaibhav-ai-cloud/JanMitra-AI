# JanMitra-AI

This repository contains the initial folder structure for the JanMitra-AI project.

## Structure
- frontend/ - Next.js web app
- backend/ - FastAPI backend
- ai-engine/ - AI services
- mobile/ - Optional mobile app
- database/ - SQL and database scripts
- docs/ - Documentation
- docker/ - Docker files
- datasets/ - AI datasets
- models/ - Trained models
- scripts/ - Utility scripts
- nginx/ - Reverse proxy configuration
- .github/ - CI/CD workflows
